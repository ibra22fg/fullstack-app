# Restaurant Recommendation System

Full-stack application with CI/CD pipeline.

## Technologies
- Frontend: HTML/CSS/JavaScript
- Backend: Node.js/Express
- Database: MySQL (Docker)
- CI/CD: Jenkins
- Containerization: Docker & Docker Compose

## Running the project
```
docker-compose up -d
```
